# 🚀 BCA Senior Guide Website

A modern, interactive portfolio-style website designed to guide BCA students from basics to career readiness.
Built with clean UI, animations, and practical resources — no fluff, only actionable content.

---

## 📌 Project Overview

This project is a **student-focused web platform** that helps BCA juniors:

* Access previous year question papers
* Understand career options after BCA
* Follow structured learning roadmaps
* Connect directly for guidance

It combines **UI design + practical content**, which makes it more valuable than a typical portfolio.

---

## 🎯 Features

* 🎨 Modern Glassmorphism UI
* ⚡ Smooth animations & hover effects
* 🧠 Career roadmap with real-world skills
* 📂 Downloadable academic resources (PDFs)
* 📱 Fully responsive design
* 🔥 Interactive elements (typing effect, 3D cube, tilt cards)

## ▶️ How to Run This Project

### Method 1 (Simple - No Setup)

1. Download or clone the repository
2. Open `index.html` in your browser
3. That’s it — project runs locally

---

### Method 2 (Recommended - Live Server)

1. Install **VS Code**
2. Install extension: **Live Server**
3. Right-click `index.html`
4. Click **"Open with Live Server"**

👉 This avoids issues like:

* Broken file paths
* PDF not loading
* JS not updating
